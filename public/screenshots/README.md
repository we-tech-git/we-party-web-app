# 📸 Screenshots

Esta pasta contém os screenshots estratégicos da plataforma We Party exibidos na landing page.

## Arquivos Necessários

Adicione os seguintes arquivos nesta pasta:

1. **feed.png** - Screenshot do feed de eventos
2. **profile.png** - Screenshot do perfil de usuário
3. **event-details.png** - Screenshot da página de detalhes do evento
4. **chat.png** - Screenshot do chat/interações
5. **create-event.png** - Screenshot da tela de criação de eventos
6. **categories.png** - Screenshot da exploração de categorias

## Especificações

- **Formato:** PNG ou JPG
- **Dimensões recomendadas:** 1200px - 1600px de largura
- **Qualidade:** Alta resolução

## Guia Completo

Consulte o arquivo `SCREENSHOTS_GUIDE.md` na raiz do projeto para instruções detalhadas sobre como capturar e otimizar os screenshots.
